Welcome to Pet Happy!

Application that simplyfy the pet adoption process.

Story

What problem does this application solve

When you go the website, you are greeted with options to sign up and serach for pet for adoption in your area.

Based on your search you will be provided with the pets that are available for adoption with pet name, about, picture and link to adopt the pet.

Technologies Used

Front End

  • View Engine: Handlebars
  • Fonts: Google Fonts
  • Color: Color Hunt
  • Icons: Font Awesome
  • Animation: Css Animation

Back End & Database

  • Language: JavaScript
  • Database: MySql

Packages & Libraries

  • bcryptjs: Password Hashing
  • Express: Web Framework
  • Express-validator: For user information validation on signup
  • Express-Sesson: User session storage.
  • Passport: User authentication
  • Sequelize: ORM
